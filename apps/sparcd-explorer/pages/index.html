SPARC'd Explorer

This focused exploration tool runs as a live marimo app. The previous GitHub Pages WASM export did not render the interactive controls, so this Pages site now acts as a handoff instead of serving a blank app.

To run it locally from the repository, use pnpm start --filter @sparcd/sparcd-explorer.

For a public deployment, run the app container from apps/sparcd-explorer/Dockerfile on a trusted HTTPS host.
